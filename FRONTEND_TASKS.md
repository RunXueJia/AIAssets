# 24小时AI增长资产引擎前端任务清单

版本：v1.0  
日期：2026-05-19  
技术栈：Vue 3 + Vite  
项目目录：frontend/admin  

## 1. 前端边界

前端只负责后台页面、交互、路由、状态和接口调用。

- 不连接 MySQL。
- 不连接 Redis。
- 不读取后端文件目录。
- 不实现抓取、LLM、视频合成等后端业务。
- 不展示 `prompt`、`model`、`temperature`、`task_id` 等技术字段给普通运营。
- 所有业务数据来自后端 API、SSE 和文件下载接口。

## 2. 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios 或 Fetch
- EventSource 或 Fetch ReadableStream
- Element Plus 或 Naive UI
- ECharts

## 4. 任务清单

### F1. 项目初始化

- 使用 Vue 3 + Vite 初始化 `frontend/admin`。
- 配置 Vue Router。
- 配置 Pinia。
- 配置 API 请求封装。
- 配置 SSE 客户端封装。
- 配置基础布局：侧边栏、顶部栏、内容区。
- 配置开发代理，只代理 `/api` 和 `/sse` 到后端。
- 配置 mock 模式，支持后端未完成时独立开发页面。

验收：

- 前端项目可独立启动。
- 后端未启动时可用 mock 数据开发。
- 不包含后端模板、数据库连接或任务逻辑。

### F2. 登录与权限

- 登录页。
- 退出登录。
- 当前用户信息获取。
- Token 保存和过期处理。
- 路由权限控制。
- 按角色控制菜单和按钮。

角色：

- 管理员
- 运营负责人
- 内容编辑
- 视频运营
- 只读查看者

验收：

- 未登录访问后台页面时跳转登录页。
- Token 过期后自动跳转登录页。
- 不同角色只能看到对应菜单和操作按钮。

### F3. 后台首页

- 展示今日生成任务数。
- 展示今日抓取任务数。
- 展示今日抓取素材数。
- 展示今日新增选题数。
- 展示今日新增脚本数。
- 展示今日合成视频数。
- 展示待审核数量。
- 展示合成失败数量。
- 展示今日导出发布包数量。

验收：

- 首页数据来自后端接口。
- 数据为空时显示友好空态。

### F4. 内容生成入口

页面字段必须语义化：

- 你想做什么内容
- 这次想讲什么主题
- 主要给谁看
- 一次生成几条
- 内容更适合哪个栏目
- 这次要生成到哪一步
- 是否现在开始

前端任务：

- 表单校验。
- 默认值设置。
- 中文业务字段映射为后端内部参数。
- 点击“开始生成内容”创建生成任务。
- 创建成功后进入生成过程页面。

验收：

- 普通运营界面不出现技术字段。
- 非开发人员能独立创建生成任务。

### F5. SSE 生成过程展示

- 接入生成任务 SSE。
- 展示阶段状态：
  - 正在创建任务
  - 正在抓取相关内容
  - 正在整理素材汇总
  - 正在生成选题
  - 正在生成脚本
  - 正在生成分镜
  - 正在生成字幕
  - 生成完成
- 展示抓取来源列表。
- 展示 LLM 文本增量。
- 展示阶段结构化结果。
- 处理 SSE 断开和重连。
- 页面刷新后恢复任务状态。

验收：

- 生成过程实时更新。
- 断线后可重新进入页面查看任务结果。
- 错误提示使用运营能理解的中文文案。

### F6. 生成任务列表

- 任务列表查询。
- 按状态筛选。
- 按创建时间筛选。
- 按创建人筛选。
- 查看任务详情。
- 取消任务。
- 重新生成。

状态：

- 等待中
- 运行中
- 成功
- 失败
- 已取消
- 重试中

验收：

- 任务状态与后端一致。
- 失败任务可查看失败原因。

### F11. 话题监控与自动任务

- 暂停监控任务。

- 删除监控任务。
