03产品定位

  • 角色边界：用户/游客与管理员，不做规划师、运营、支付和订单

05技术架构

三端协作与外部能力接入

FastAPI 作为统一后端，用户端与管理端共享认证和记录能力。

  • Web 用户端：Vue 3 / Vite / Pinia / Element Plus
  • 后端 API：FastAPI / SQLAlchemy 2 / Pydantic v2
  • MySQL：用户、记录、快照、日志、配置
  • 外部服务：高德、天气、Tavily、OpenAI-compatible LLM
