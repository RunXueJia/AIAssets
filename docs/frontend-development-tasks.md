# 前端开发任务列表

技术栈建议：Vue 3、Vite、JavaScript、Pinia、Vue Router、Element Plus、Axios、SCSS。项目可拆为 `apps/admin` 管理后台和 `apps/web` 公开站。

## 1. 基础工程

- [ ] 初始化 `apps/admin` Vue3 管理后台。
- [ ] 初始化 `apps/web` Vue3 公开站。
- [ ] 配置 Vite、ESLint、Stylelint、Prettier。
- [ ] 配置 Element Plus、自动导入、路由、Pinia。
- [ ] 封装 Axios：baseURL、token、错误处理、401 处理、统一响应解包。
- [ ] 封装 SSE 流式请求工具：使用 `fetch` + `ReadableStream` 处理 POST 流式接口，解析 OpenAI 标准 `data:` chunk 和 `[DONE]`。
- [ ] 建立 API 模块目录：按业务模块拆分。
- [ ] 建立权限常量和路由元信息。
- [ ] 建立基础布局：登录页、后台主布局、公开站布局。

## 2. 管理后台通用能力

- [ ] 登录页：账号、密码、验证码预留、错误提示。
- [ ] 顶部栏：当前用户、退出、角色展示。
- [ ] 侧边栏：按权限渲染菜单。
- [ ] 路由守卫：未登录跳登录、无权限跳 403。
- [ ] 按钮权限：封装 `v-permission` 或 `usePermission`。
- [ ] 通用列表页能力：分页、搜索、筛选、排序、加载态、空态。
- [ ] 通用表单弹窗：新增、编辑、校验、提交态。
- [ ] 通用状态标签：内容状态、任务状态、发布状态。
- [ ] 通用文件预览：视频、图片、PDF、Markdown。

## 3. 系统与权限页面

- [ ] 用户管理：列表、新增、编辑、禁用、重置密码、分配角色。
- [ ] 角色管理：列表、新增、编辑、分配权限。
- [ ] 权限点展示：按模块树形展示。
- [ ] 操作审计：筛选、查看详情。
- [ ] 系统配置：对象存储、任务默认配置、密钥脱敏展示。
- [ ] LLM 供应商配置：baseURL、API Key 脱敏、启用状态、健康检查。
- [ ] LLM 模型配置：模型名、用途、温度、最大 token、上下文长度、成本单价。
- [ ] Prompt 模板管理：场景、版本、system prompt、user prompt、变量说明、JSON Schema。
- [ ] Prompt 调试页：选择模板和模型，填写变量，实时展示 SSE 增量输出、原始 chunk、完成后的解析结果。
- [ ] LLM 调用日志：任务类型、模型、token、耗时、费用估算、状态、错误详情。
- [ ] LLM 成本看板：按日期、模型、栏目、任务类型查看调用量和成本。

## 4. 内容配置页面

- [ ] 内容方向列表：搜索、状态筛选、启用、停用。
- [ ] 内容方向表单：名称、描述、受众、风格、禁止话题、每日目标产量、审核人。
- [ ] 栏目列表：按内容方向筛选。
- [ ] 栏目表单：名称、描述、默认时长、默认视频模板、默认图文模板。
- [ ] 视频模板管理：模板类型、分辨率、预览图、状态。
- [ ] 图文模板管理：模板名称、布局、SEO 配置、状态。
- [ ] 平台配置管理：标题长度、简介长度、标签限制、状态。

## 5. 内容生产页面

- [ ] 选题列表：栏目、状态、日期、关键词筛选。
- [ ] 选题详情：标题、副标题、受众、角度、关键词、生成原因、重复提示。
- [ ] 选题操作：锁定、驳回、生成脚本、批量生成。
- [ ] 生成参数面板：选择模型、Prompt 模板版本、生成数量、温度和关键词种子。
- [ ] 流式生成面板：生成中实时展示 LLM 增量文本、chunk 数量、首 token 耗时、总耗时和中断状态。
- [ ] 脚本列表：按选题、栏目、版本、审核状态筛选。
- [ ] 脚本详情编辑：钩子、主体、结尾、平台标题、简介、标签、风险标记。
- [ ] 脚本版本对比：查看历史版本和当前版本差异。
- [ ] 分镜编辑器：镜头序号、时长、旁白、字幕、画面类型、素材建议、动效建议。
- [ ] 质量检查面板：风险词、可疑事实、夸张承诺、高风险主题。
- [ ] 重新生成入口：选题、脚本、分镜分别支持重新生成。
- [ ] 生成记录抽屉：展示本内容关联的 LLM 调用日志、Prompt 版本、OpenAI SSE 原始 chunk、聚合输出、解析结果。

## 6. 审核工作台

- [ ] 待审核列表：选题、脚本、分镜、视频、图文混合或分 Tab。
- [ ] 审核详情：内容正文、分镜、字幕、来源说明、风险提示。
- [ ] 审核动作：通过、驳回、修改后通过、重新生成。
- [ ] 驳回原因弹窗。
- [ ] 修改后通过保存版本。
- [ ] 审核记录时间线。
- [ ] 待处理数量提醒。

## 7. 视频与资产页面

- [ ] 视频资产列表：状态、栏目、日期、模板筛选。
- [ ] 视频详情：播放器、封面、分辨率、时长、渲染日志。
- [ ] 视频操作：下载、重新合成、重新生成封面。
- [ ] 知识卡片列表：按内容筛选、预览、下载 PNG。
- [ ] 资料包列表：类型、标题、状态、下载、重新生成。
- [ ] 文件上传/预览组件：用于模板素材和补充资料。

## 8. 发布队列页面

- [ ] 待发布列表：平台、栏目、状态、日期筛选。
- [ ] 发布包详情：视频、封面、标题、简介、标签、置顶评论、图文版。
- [ ] 发布包下载。
- [ ] 记录平台链接。
- [ ] 发布状态变更：待发布、已发布、已下线。
- [ ] 按平台聚合查看发布状态。

## 9. 数据看板与日报页面

- [ ] 今日总览：选题、脚本、视频、图文页、知识卡片、资料包、发布、访问、下载、线索。
- [ ] 任务运行：成功率、失败率、平均耗时、失败原因列表。
- [ ] 栏目表现：产出数量、访问、下载、线索。
- [ ] 内容资产规模：累计视频、页面、资料包、线索。
- [ ] 每日报告列表：日期、生成状态、查看、导出。
- [ ] 日报详情：Markdown 渲染、导出按钮。

## 10. 公开站页面

- [ ] 首页：最新视频、热门栏目、热门资料包、推荐专题、订阅入口。
- [ ] 栏目页：栏目介绍、内容列表、分页。
- [ ] 视频详情页：视频播放器、标题、摘要、文字稿、相关推荐、转化入口。
- [ ] 图文详情页：正文、操作步骤、使用场景、注意事项、资料下载入口。
- [ ] 专题页：专题介绍、关联内容、资料包。
- [ ] 资料下载页：资料说明、表单、用户授权提示、下载结果。
- [ ] 工具推荐页：工具清单、适用场景、关联内容。
- [ ] 订阅页：邮箱或联系方式提交。
- [ ] 移动端适配和基础 SEO meta。

## 11. 前端验收

- [ ] 管理后台能完整跑通 MVP 操作链路。
- [ ] 普通未登录用户只能访问公开站。
- [ ] 后台不同角色看到的菜单和按钮符合权限。
- [ ] 所有列表支持分页、筛选、加载态、空态和错误态。
- [ ] 表单字段和后端校验保持一致。
- [ ] 公开站移动端可浏览，内容页有转化入口。
- [ ] 视频、图片、PDF、Markdown 可预览或下载。
- [ ] 关键页面有基础 E2E 用例：登录、审核、发布包下载、资料下载表单。
