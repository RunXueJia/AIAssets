# 后端任务清单

## 1. 基础工程

- [ ] 初始化 FastAPI 项目结构。
- [ ] 配置 pyproject.toml。
- [ ] 配置环境变量和 settings。
- [ ] 配置后端本地默认端口为 `3002`。
- [ ] 接入 MySQL。
- [ ] 按 `docs/db/mysql_schema.md` 落地 MySQL 表结构和 SQLAlchemy 模型。
- [ ] 接入 SQLAlchemy 2 异步会话。
- [ ] 配置 Alembic 迁移。
- [ ] 配置统一异常处理。
- [ ] 配置统一响应结构。
- [ ] 按 `docs/api/frontend_backend_integration.md` 实现前后端联调接口。
- [ ] 配置请求日志和链路 ID。
- [ ] 配置 CORS。
- [ ] 配置 ruff、mypy、pytest。

## 2. 用户与管理员

- [ ] 用户注册登录。
- [ ] 游客临时会话。
- [ ] JWT 鉴权。
- [ ] 管理员登录。
- [ ] 初始化默认管理员账号：用户名 `admin`，初始密码 `admin123456`，密码入库必须哈希存储。
- [ ] 管理员用户列表。
- [ ] 管理员禁用/启用用户。

## 3. 生成记录模块

- [ ] 创建生成记录。
- [ ] 保存用户输入快照。
- [ ] 查询用户自己的生成记录。
- [ ] 查询生成记录详情。
- [ ] 保存最终生成结果。
- [ ] 保存生成状态：pending、streaming、completed、failed。
- [ ] 管理端查询全部生成记录。
- [ ] 管理端查看错误详情。
- [ ] 管理端重试失败记录。

- [ ] 设计 TripPlanningRequest Schema。

- [ ] 实现 SSE 或 WebSocket 接口。
